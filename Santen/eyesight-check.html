Проверка зрения у детей

Способность различать с расстояния 5 м разрывы в кольцах, сгруппированных на левом рисунке, соответствует остроте зрения вдаль:

1,0 — для внутреннего,

0,3 — для среднего.

0,1 — для наружного кольца

То же на рисунке справа:

2,0 — для внутреннего,

0,5 — для среднего,

0,2 — для наружного кольца.
